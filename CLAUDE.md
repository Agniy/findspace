# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
go build -o findspace .      # сборка бинарника
go run main.go [path] [mb]   # запуск без сборки
go mod tidy                  # обновить зависимости
```

## Usage

```
findspace [path] [min_mb]
```

- `path` — директория для обхода (по умолчанию `.`)
- `min_mb` — скрыть директории меньше N MB (по умолчанию `0` — показывать всё)

## Architecture

Весь проект — один файл `main.go`. Поток выполнения:

1. `main` парсит аргументы, устанавливает пакетную переменную `minSize`, вызывает `buildTree(abs, 2)`
2. `buildTree` рекурсивно строит дерево `DirNode` глубиной 3 уровня (depth 2 → 1 → 0):
   - На каждом уровне запускает горутину на каждую поддиректорию
   - На `depth == 0` вызывает `calcSize` (полный рекурсивный обход без создания узлов)
   - Дочерние узлы меньше `minSize` не добавляются в `Children`, но их размер суммируется в родителе
3. `printTree` рекурсивно выводит готовое дерево с символами `├──` / `└──`

### Важное про семафор

`sem` — буферизованный канал размером `NumCPU*8`. Он захватывается **только на время самого I/O-вызова** (`os.ReadDir` / `calcSize`) и сразу освобождается. Держать семафор во время `wg.Wait()` или рекурсивного `buildTree` вызывает дедлок: родительская горутина ждёт детей, дети не могут получить слот семафора.
